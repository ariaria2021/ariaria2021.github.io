## 概要

「Boulder Defense」は、物理演算を取り入れたタワーディフェンスゲームです。
敵は単に道を歩くだけでなく、**坂道を転がり、加速し、勢いよく拠点に突っ込んできます**。

## 特徴（開発中）

- **物理挙動**: 坂道での加速、慣性による移動。
- **多彩な敵**: 通常の岩に加え、金属製の岩やマグマを纏った岩など。
- **戦略的防衛**:
    - **砲台**: 物理的な衝撃で岩を砕く。
    - **溶解液**: 金属を溶かす。
    - **消滅装置**: 高コストだが一撃必殺。
- **トラップ**: 落とし穴やジャンプ台で敵の進行を物理的に阻害。

## 技術スタック

- **Core**: TypeScript + HTML5 Canvas API
- **Tooling**: Vite
- **Infrastructure**: GitHub Actions + GitHub Pages
